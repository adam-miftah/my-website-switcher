Saklar Kontrol Website

Gunakan saklar di bawah ini untuk mengubah status website utama Anda. PENTING: Setelah mengubah status, Anda harus **menyalin JSON yang dihasilkan** lalu **mengedit dan meng-commit file `status.json` di repository Git Vercel Anda**. Vercel akan otomatis mendeploy ulang.

AKTIF

ALUR KERJA PENTING:

  1. Ubah saklar di atas.
  2. Salin teks JSON dari kotak di atas.
  3. Buka **repository Git** proyek saklar Anda (misalnya di GitHub/GitLab).
  4. Navigasi ke file public/status.json.
  5. Edit file tersebut, **hapus konten lama dan tempelkan konten JSON yang baru Anda salin**.
  6. **Commit perubahan** ke repository Git Anda.
  7. Vercel akan secara otomatis mendeploy ulang proyek saklar Anda. Setelah deployment Vercel selesai, website utama Anda (di Hostinger) akan membaca status baru ini dalam beberapa detik.
